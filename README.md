# Legacy-Console-Text

Legacy Console Text (LCT) は、Java Edition の各テキストを Legacy Console Edition のものに変更するリソースパックです。

例えば、ブロックやアイテム、MOBなどの名前を以下のように変更します。

- ネザーラック -> 暗黒石
- スケルトン -> ガイコツ
- ダメージ軽減 -> 防護

> [!NOTE]
> このパックは Minecraft 1.17 以降のバージョン向けに作成されているため、1.16以前のバージョンではうまく動作しない可能性があります。
